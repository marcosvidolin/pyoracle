# pyoracle

Ubuntu image with Python and Oracle Instant Client

## How to

```bash
docker run -p 5000:5000 -e "DB_USERNAME=myuser" -e "DB_PASSWORD=mypass" -e "DB_CONNECT_DESCRIPTOR=localhost:1521/foo" vidolin/pyoracle:latest
```
